TEAM FINDER

Team Finder es un proyecto diseñado para facilitar la creación de equipos de trabajo eficientes, basándose en las habilidades técnicas y blandas de cada integrante. Su objetivo principal es ayudar a empresas de tecnología a conformar equipos equilibrados y eficientes, mejorar la integración y optimizar la colaboración interna.

Formato: Aplicación web

Tecnologías: React, Node.js, Prisma, APIs, JavaScript, HTML, CSS

Herramientas de gestión: Linear, GitHub, Discord, diagramas PERT y Gantt, etc...
